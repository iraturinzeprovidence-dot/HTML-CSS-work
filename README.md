# Mpuza Login Webpage

##  Overview
This project is a **single HTML & CSS login page** for the application **Mpuza**.  
It demonstrates a simple, professional interface for user authentication.

---

##  Page Structure
- **Header**  
  - Red background with the text **Mpuza**.

- **Login Form**  
  - Email input field with icon.  
  - Password input field with icon and eye icon (static).  
  - "Remember me" checkbox.  
  - Rounded **Sign In** button styled in blue.  

- **Footer**  
  - Blue background with the text **Kigali-Rwanda**.

---

##  Technologies
- **HTML5** for structure  
- **CSS3** for styling  

---

##  How to View
1. Save the file as `login.html`.  
2. Place the required image assets (`Capture.PNG`, `carbon--email.png`, `carbon--password.png`, `mdi--eye-outline.png`) in the same folder.  
3. Open `login.html` in any modern web browser.
